# Vehicle Rental System 

**Live URL:** https://l2-b6-a2-six.vercel.app/

---

## Project Overview
A full-featured backend API for managing a vehicle rental system.  
Supports vehicles, customers, bookings, authentication, and automated business rules.

**Core Features:**
- Role-based access (Admin & Customer)
- Vehicle inventory management with availability tracking
- Customer account management
- Booking management with price calculation
- Booking cancellation & return logic
- Auto-return of bookings after end date
- Secure JWT authentication & password hashing
- Modular & scalable code architecture

---

##  Technology Stack
- **Backend:** Node.js + TypeScript  
- **Web Framework:** Express.js  
- **Database:** PostgreSQL  
- **Authentication:** JWT (jsonwebtoken)  
- **Password Security:** bcrypt  
- **Cron Jobs:** node-cron  
- **Other Tools:** cors, dotenv
